# CDriveMaster v1 冒烟测试人工验收清单

## 启动与调度
- [ ] 程序双击启动不崩溃。
- [ ] 点击当前 UI 上用于开始扫描的按钮后，进度条正常跑动。
- [ ] 点击“取消”后扫描应尽快中断，UI 不假死。

## 数据分析
- [ ] Downloads / Desktop / Documents / Temp 四类目标均被扫描。
- [ ] 四个分类摘要卡片数据正常。
- [ ] Top 20 大文件正常显示。
- [ ] Top 20 大目录正常显示。

## 异常防御
- [ ] “跳过路径 / 拦截记录”页签中应能看到被跳过项样本。
- [ ] 如果当前实现保留异常原因，则应可看到访问拒绝类原因（如 UnauthorizedAccessException 或 Access Denied）。

## 受控清理 A：回收站
- [ ] 点击“清空回收站”后，若当前实现启用了系统确认框，则应能正常弹出。
- [ ] 清空成功后，回收站占用显示应刷新为 0 或明显下降。

## 受控清理 B：Temp
- [ ] 点击当前 UI 上用于清理 Temp 的按钮后，清理流程正常完成。
- [ ] 锁定跳过、保护期跳过等漏斗统计应能正常显示。
- [ ] 即使有部分文件删不掉，整体流程也不能崩溃。

## 手动删除大文件
- [ ] 在大文件列表中，通过当前实现的右键菜单执行“移至回收站”。
- [ ] 二次确认后，该文件应从当前列表中移除，或在刷新后消失。
- [ ] 回收站容量应同步增加。
