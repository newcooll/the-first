# C盘管理大师 v1

## 目标
做一个 Windows 桌面工具，用于分析 C 盘空间占用，并提供低风险清理功能。

## v1 功能
- 扫描 C 盘主要用户目录空间占用
- 找出最大文件和最大文件夹
- 显示回收站、Temp、Downloads 占用
- 支持清空回收站
- 支持清理 Temp
- 支持删除用户手动选中的文件
2. v1 明确不做什么
## v1 不做
- 不删除 Windows 系统目录
- 不碰 WinSxS
- 不改注册表
- 不做“系统加速”
- 不做“一键优化全盘”
3. 用户流程
## 用户流程
1. 打开软件
2. 点击“开始扫描”
3. 查看占用最大的目录/文件
4. 勾选低风险清理项
5. 二次确认后执行
4. 成功标准
## 成功标准
- 能正确显示 C 盘总空间、可用空间、已用空间
- 能列出前 50 个最大文件
- 能统计 Downloads、Desktop、Documents、Temp 占用
- 清空回收站功能可用
- 清理 Temp 功能可用
为什么这是第一步
因为你现在缺的不是代码，而是约束

如果没有这一步，后面会出现这些问题：

扫描逻辑和清理逻辑混在一起
UI 做了半天，核心功能没定
权限和风险模型完全失控
项目名字很大，第一版却什么都想做，最后什么都做不稳

先定边界，再写扫描器。
